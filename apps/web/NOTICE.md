## Trezo – Admin Dashboard Template
A base visual deste front (em especial `src/app/globals.css` — sistema de tokens
Tailwind — e o padrão de layout do shell) deriva do template comercial **Trezo**
(variação React/Next + Tailwind).

- Uso licenciado sob **Extended License** (cobre SaaS multitenant com usuários
  pagantes). Ver `docs/spec/10-ux-ui-design-system.md` §8.
- Não redistribuir o template como template; o entregável permitido é o produto
  final (VETAPP).
- Apenas a variação `react-nextjs-tailwindcss` é utilizada; as demais variações do
  pacote não são versionadas neste repositório (higienização — doc 10 §6).
